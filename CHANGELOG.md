# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-12-12

### Added
- Initial release with 9 philosophical thinking tools
- Stoic framework: dichotomy of control, premeditation, memento mori
- Cognitive framework: bias scanning with Kahneman's System 1/2
- Mindfulness framework: STOP practice for mindful pausing
- Strategic framework: Musashi's timing and Five Rings analysis
- Meta tools: Socratic clarification and framework recommendation
- Embedded wisdom quotes from primary sources
- Claude Desktop and Claude Code configuration support
